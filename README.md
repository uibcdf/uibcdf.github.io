# UIBCDF Lab Website

    Website for the **Unidad de Investigación en Biología Computacional y Diseño de Fármacos (UIBCDF)**.

Hosted on GitHub Pages with the custom domain **www.uibcdf.org**.

    **Live site:** https://www.uibcdf.org/

    ## What’s in this repo
    - `web/` — Astro site source
    - `devguide/` — project checkpoint (concept, IA, content model, decisions, deploy, workflow)
    - `resources/` — logos, images, diagrams used by the site

    ## MVP navigation
    - Research
- People
- Publications
- Software
- Updates
- Contact

    ## Content editing (Markdown-first)
    Content lives under `web/src/content/` (see `devguide/CONTENT_MODEL.md`).
    Editors can add/update entries via GitHub by editing the Markdown files.

    ## Development (local)
    This repo is designed to be developed locally (often with Codex) after cloning.
    See `devguide/06_workflow_with_codex.md`.

    ## Decisions & scope
    - Decisions: `devguide/DECISIONS.md`
    - MVP scope: `devguide/01_mvp_scope.md`
    - IA / sitemap: `devguide/02_information_architecture.md`
    - Brand & tone: `devguide/BRAND.md`
